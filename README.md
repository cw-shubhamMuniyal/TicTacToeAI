# TicTacToe
You can play Tic Tac Toe in 2 modes
- Human vs Human Mode
- Human vs Ai Mode


# Tech Stack
- Html, CSS for design
- P5.js for sketching


# Implementation of AI mode
I have used Min Max algorithm to let AI make strategic moves instead of random moves.
